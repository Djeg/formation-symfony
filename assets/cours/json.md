# Le Format JSON

JSON est un format de représentation de données basé sur la syntax
javascript :

```json
{
  // Objet
  "maClef": {
    // un second objet
    "autreClef": 10, // number
    "autreClef2": false, // boolean
    "autreClef3": "John", // string
    "autreClef4": [10, 15, 8, 63] // array
  }
}
```
